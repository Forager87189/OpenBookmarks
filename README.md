# 📚 OpenBookmarks

![alt text](image.png)

**OpenBookmarks**는 다양한 주제의 웹사이트 링크를 카테고리별로 정리하고, 누구나 자유롭게 기여할 수 있도록 만든 오픈소스 북마크 웹 애플리케이션입니다.

---

## 🚀 프로젝트 소개

- 사용자가 북마크를 카테고리별로 추가 및 관리할 수 있습니다.
- 깔끔한 UI와 직관적인 인터페이스를 제공합니다.
- 누구나 쉽게 기여하고 새로운 북마크를 추가할 수 있도록 설계되었습니다.

---

## 🛠️ 기술 스택

- **Frontend**: React
- **Backend**: Spring Boot
- **Data**: MySQL
- **버전 관리**: Git & GitHub
